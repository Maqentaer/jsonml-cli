#jsonml-cli

Parse HTML and convert to JSONML

##Usage

  jsonml.js [OPTIONS]

##Options

  -o, --out PATH         Output file
  -i, --in PATH          Input file
  -u, --url URL          Input URL
  -v, --version          Display the current version
  -h, --help             Display help and usage details

#Without -i, --in and -u, --url
  input from stdin

#Without -o, --out
  output to stdout
